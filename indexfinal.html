We used the traffic accidents dataset from the cityofchicago.org available through Socrata Open Data API (https://dev.socrata.com/consumers/getting-started.html).
We also used Chicago neighborhood data from available from Kaggle (https://www.kaggle.com/doyouevendata/chicago-community-areas-geojson) for some of the visualizations.
